Click on any cell in Google Sheets and type =Intrinio to see a list of all Intrinio functions. Once you select a function, the required parameters and an example will show you how to retrieve data.

There are three functions that are enabled with the Intrinio Google Sheets add-on, allowing you to pull in thousands of data points per company. These functions include:

  • =IntrinioDataPoint
  • =IntrinioFundamentals
  • =IntrinioFinancials

=IntrinioDataPoint(ticker, item)

  • ticker - The company's ticker symbol (i.e. "AAPL")
  • item - The data field requested for the company. A list of all tags can be found here (i.e. "trailing_dividend_yield" returns the current dividend yield, "current_yr_ave_eps_est" returns the current fiscal year average Wall Street consensus EPS estimate)
Example: =IntrinioDataPoint("GOOGL","close_price") -> $538.65 (The last close price for Google Inc was $538.65)

=IntrinioFundamentals(ticker,statement,period_type,sequence,item)

  • ticker - The company's ticker symbol (i.e. "AMZN")
  • statement - The financial statement selected ("income_statement","balance_sheet","cash_flow_statement","calculations")
  • period_type - The period type ("FY","QTR","TTM","YTD")
  • sequence - The sequence order of the fundamental from newest to oldest (0..last available)
  • item - The item you are selecting (i.e. "fiscal_year" returns 2014, "fiscal_period" returns "FY", "end_date" returns the last date of the period)
Example: =IntrinioFundamentals("GOOGL","income_statement","FY",0,"fiscal_year") -> 2014 (The last fiscal year for Google Inc was 2014)

=IntrinioFinancials(ticker,statement,fiscal_year,fiscal_period,tag,rounding)

  • ticker - The company's ticker symbol (i.e. "FB")
  • statement - The statement selected ("income_statement","balance_sheet","cash_flow_statement","calculations")
  • fiscal_year - The selected fiscal year for the chosen statement (i.e. 2014, 2013, 2012) or the sequence order of the fundamental from newest to oldest (0..last available)
  • fiscal_period - The selected fiscal period for the chosen statement ("FY","Q1","Q2","Q3","Q4","Q1TTM","Q2TTM","Q3TTM","Q2YTD","Q3YTD") or the period type ("FY","QTR","TTM","YTD")
  • tag - The selected tag within the statement. A list of all tags can be found here (i.e. "totalrevenue", "netppe", "totalequity","purchaseofplantpropertyandequipment", "netchangeincash")
  • rounding - Round the value (blank or "A" for actuals, "K" for thousands, "M" for millions, "B" for billions)
Example: =IntrinioFinancials("GOOGL","income_statement",0,"FY","totalrevenue","M") -> $66,001 (The total revenues for Google Inc in the most recent fiscal year (2014) was $66,001 million)
